# 小红书爆款标题 Skill

面向小红书平台，批量产出高点击、高完读率的标题方案，覆盖“反差/悬念/数字/痛点-结果/人群标签/季节热点”等核心玩法，并内置合规与长度控制。

## Quick Start

请在任务中尽量给出：话题、目标人群、核心卖点/痛点、要包含的关键词、风格偏好。

示例指令：
```
帮我为「油皮夏季防晒」生成20个小红书爆款标题，
受众：学生党/通勤族，关键词：清爽、不拔干、平价，
风格：真实测评+轻松口语，长度约20-28字，适当加1-2个emoji。
```

## 产出要求

1) 标题数量：默认 15–20 条，可调整 `count`。
2) 长度控制：16–28 字为宜（中文字符计），支持 `length_limit` 自定义。
3) 角度覆盖：每条标注角度标签（如【反差】【悬念】【数字】【痛点-结果】【人群】【季节】等）。
4) 表情与口语：适度 1–2 个 emoji；避免密集符号堆砌；贴近口语、具体化。
5) 关键词注入：尽量自然包含用户给定关键词；避免生硬堆砌。
6) 合规约束：避用极限词/绝对化承诺/违规功效词；规避导流、敏感对比与医疗功效断言。
7) A/B 测试友好：同一卖点输出多种钩子变体（悬念版/数字版/结果导向版等）。

## 角度库（择优混用）

- 反差/反常识：用“和认知相反”的点吸引点击。
- 悬念/槽点：制造未解之问、留白或对比冲突。
- 数字/清单：N个、TOP清单、预算档位、对比表述。
- 痛点→结果：直击痛点，承诺可验证的具体结果（避免绝对化）。
- 人群/场景：按身份、肤质/发质、预算、地域、季节、时间段等切分。
- 具体利益点：可量化、可感知的微差异（“不搓泥”“不拔干”“不显毛孔”）。
- 稀缺/时令/节点：618、双11、开学季、梅雨季、通勤、健身季等。
- 提问/共鸣：把用户内心独白写成问题。

## 输出格式

请按如下格式输出，便于快速评审与复制：

```
1. 标题文案 【角度】（理由/钩子简述）
2. 标题文案 【角度】（理由/钩子简述）
...

可选：
- 主题标签：#关键词1 #关键词2 #场景
- 备选短词（可替换）：词A / 词B / 词C
```

## 合规与风格校验清单

- 长度在目标区间内（16–28 字）。
- 避免“最、第一、百分百、永久”等极限/夸张承诺。
- 不做医疗/功效断言；避免引战、攀比、人身攻击。
- 不导流站外；不含联系方式、价格跳转诱导。
- 用词具体、接地气，避免堆叠形容词和营销腔。

## 可用变量（在用户指令中可指定）

- `topic`：主题/品类（如：油皮防晒、干皮粉底、山姆囤货）。
- `audience`：目标人群（学生党、通勤族、黄黑皮、敏感肌）。
- `scene`：使用场景（梅雨季、通勤地铁、海边旅行、开学季）。
- `pain_points`：主要痛点（搓泥、闷痘、出油、卡粉、不持久）。
- `benefits`：期望收益（清爽、提亮、持久、遮瑕、轻薄）。
- `keywords`：必须包含的词（平价、不拔干、不假白）。
- `style`：文风（真实测评、轻松口语、避雷科普、清单种草）。
- `count`：标题数量，默认 20。
- `length_limit`：目标字数区间，默认 16–28。
- `emoji_policy`：无/适度(默认)/强调。

## 生成流程（给模型的操作指引）

1. 复述输入：人群/场景/痛点/关键词，用 1 行确认理解方向。
2. 提取钩子：从痛点与利益点各挑 2–3 个可证实的钩子。
3. 角度编排：每个钩子做 2–3 种变体（悬念/数字/结果/人群）。
4. 产出标题：生成 `count` 条，控制在 `length_limit` 内，注入关键词。
5. 标注角度与简短理由，便于筛选与二次改写。
6. 自检：跑“合规与风格校验清单”，删除不合规项再补齐数量。

## 示例

输入：
```
topic: 油皮夏季防晒
audience: 学生党/通勤族（黄黑皮友好）
keywords: 清爽、不拔干、平价
style: 真实测评 + 轻松口语
```

可能输出（节选）：
```
1. 一整天不闷不泛油？这支平价防晒我服了【痛点-结果】（直击“出油闷痘”）
2. 黄黑皮不灰不假白，走廊灯下也稳【反差】（反常识场景）
3. 平价但不糊脸！我通勤流汗也不拔干【场景/人群】（通勤流汗）
4. 压马路3小时，还能清爽不搓泥？答案在这【悬念】（制造未解）
5. 油皮别再踩雷了：我回购的3支清爽线【数字/清单】（TOP清单）
```
